# StackChan Recovery Firmware

**ARCHIVAL & RECOVERY VERSION**

This repository serves as an archival recovery of the current StackChan firmware version with all 12 emotion faces including the sinister rat face emotion system. This is a preserved snapshot for recovery and reference purposes.

## 🎯 Purpose

- **Recovery Archive**: Preserved working firmware snapshot
- **Reference Version**: Complete emotion system with sinister rat face base
- **Backup Repository**: Separate from main development repository
- **Emergency Restore**: Quick access to known-good firmware state

## 📦 What's Included

### Emotion Rendering System
- **12 Complete Emotion Faces** with sinister rat face base:
  - Angry: Extra menacing teeth, angry eyebrows, red eyes
  - Happy: Gentler teeth, arched eyebrows, yellow eyes, gentle whiskers
  - Sad/Neutral: Standard menacing teeth, subtle eyebrows
  - Thinking: Furrowed eyebrows, yellow eyes
  - Speaking: Green eyes, animated mouth with speech patterns
  - Listening: Blue eyes, attentive expression
  - Looking: Large cyan eyes, camera-assist style
  - Found: Discovery-oriented features on evil base
  - Error: Red eyes, frown expression
  - Sleep: Closed eyes, sleep mode with "Zz"
  - Rat: Enhanced whiskers, brown eyes, rat-specific features
  - Sinister: Maximum evil features with red eyes and sinister blinking (DEFAULT)

### Base Evil Features (Applied to ALL emotions)
- Evil shield/buckler design as face border
- Sharp, aggressive ears with twitch animation
- Evil shield symbol on forehead
- Menacing teeth (intensity varies by emotion)
- Evil whiskers (varying style by emotion)
- Sinister eyebrows (varying shape by emotion)
